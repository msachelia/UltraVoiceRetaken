# UltraVoice Retaken (version 1.4.0)

UltraVoice Retaken is a fork of the UltraVoice mod for ULTRAKILL that adds custom voice lines to some enemies, giving them more personality during combat and cutscenes.

This mod has been picked by achelia since the lead developer of the mod, mel, announced she won't work on the project anymore. This is done with her permission and anything before version 1.3.0 belongs to her.

Links to the original mod page and the original GitHub repo: [Thunderstore Page](https://thunderstore.io/c/ultrakill/p/mof_n_co/UltraVoice/), [GitHub Repo](https://github.com/mof33/UltraVoice)

This mod is **NOT FINISHED**. It will continue to expand with more enemies and voice content as development continues. Please report any bugs you see or suggestions you have to the [UltraVoice Retaken Discord Server](https://discord.gg/fW7NvE5WDM).

## Installation

1. Install BepInEx for ULTRAKILL
2. Download the latest release of UltraVoice Retaken
3. Place `UltraVoice.dll` in your plugins folder
4. Launch the game
