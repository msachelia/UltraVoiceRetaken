# UltraVoice Retaken (version 1.3.0)

UltraVoice Retaken is a fork of UltraVoice mod for ULTRAKILL that adds custom voice lines to some enemies, giving them more personality during combat and cutscenes.

This mod has been picked by achelia since the lead developer of the mod, mel, announced she won't work on the project anymore. This is done with her permission and anything before version 1.3.0 belongs to her.

Links to the original mod page and the original GitHub repo: [Thunderstore Page](https://thunderstore.io/c/ultrakill/p/mof_n_co/UltraVoice/), [GitHub Repo](https://github.com/mof33/UltraVoice)

## Current Character Roster

This mod currently adds voice lines to:

* Malicious Faces
* Swordsmachines (including Agony and Tundra)
* Cerberi
* V2 (both fights, masc and fem versions)
* Streetcleaners
* Mindflayers (both masc and fem)
* Virtues
* Sentries
* Ferrymen (including Agonis and Rudraksha)
* Guttermen
* Guttertanks
* Earthmover
* Providences
* Mirror Reapers
* Geryon
* Leviathan
* Insurrectionists
* Idols
* Hideous Masses

This mod currently adds ambient non-dialogue to:

* Filth
* Strays
* Schisms
* Soldiers
* Stalkers
* Mannequins

## Installation

1. Install BepInEx for ULTRAKILL
2. Download the latest release of UltraVoice Retaken
3. Place `UltraVoiceRetaken.dll` in your plugins folder
